## Git Aliases

| Алиас | Команда | Описание |
|-------|----------|-----------|
| `st` | `status` | Показывает текущее состояние репозитория |
| `ci` | `commit` | Создание коммита |
| `co` | `checkout` | Переключение между ветками |
| `br` | `branch` | Управление ветками |
| `df` | `diff` | Сравнение изменений |
| `lg` | `log --oneline --decorate --all --graph` | Графическая история коммитов |
| `recent` | `log --oneline -n 5` | Показывает последние 5 коммитов |
